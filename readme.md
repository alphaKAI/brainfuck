#Brainfuck
A minimal Brainfuck Implementation in D.  
  
#Requirements
* Latest DMD(The D Compiler.)
* Latest DUB(A D's package manager and build tool.)

#Usage
##Run as repl

```zsh
% ./brainfuck  
```

##Run with exists file

```zsh
% ./brainfuck foo.b  
```
